Graph Theory [Continued]

Subgraphs

Def: A graph G1 = (V1, E1) is said to be a subgraph of a graph G2 = (V2, E2) if V1 subset V2 and E1 subset E2.

Weighted Graphs

  • Sometimes, we use edges to denote a connection between a pair of nodes where the connection has a capacity or weight.
    • Example: resistance of a wire between two terminals, the distance of highway between two cities.

Adjacency Matrices

  • There are many ways to represent a graph. We've already seen
    • Drawing the graph
    • Representing it with sets V,E

Def: Given an n-node graph G = (V,E) where V = {v1, v2, ..., vn}, the adjacency matrix for G is the n x n matrix AG = {aij} where

aij = 1 if {vi, vj} \exists E. 0 otherwise

For weighted graphs, replace 1 with the weight w.
